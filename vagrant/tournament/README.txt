to use this program, follow these steps:


1) Navigate to the tournament folder
2) log into sql (psql)
3) build the database (and connect) by running tournament.sql (\i tournament.sql) to create the database
4) quit the database (\q)
5) run tournament_test.py to ensure tournament.py is working correctly (python tournament_test.py)
6) tournament.py is ready to be integrated into your project!
